# Frontend

This folder contains a placeholder UI concept for a future web dashboard.

## Purpose

- Illustrate the visual experience for secret inventory management.
- Show how users will upload `.env` files and view masked secret previews.
- Present role-based actions and share token workflows.

## Notes

The current demo is implemented in `demo/app.py` using Gradio for fast deployment.
A production frontend can be built as a React or Vue app that calls `/upload-env`, `/secrets`, and `/share-token`.
